GIT CHEATSHEET

Shambhavi Sachid

git config

Set up user name
git config --global user.name "Your name"

Set up user email
git config --global user.email "your_email@whatever.com"

Verify user name
git config --get user.name

Verify user email
git config --get user.email

Git Help

Viewing the list of commands
git help

Getting a command info
git help command_name

Getting info about config commandgit help config

Staging and Committing

Create an empty repository
git init

Add hello.rb file to the staging area
git add hello.rb
git add .
git add *.txt

Record changes to the repository
git commit -m "commit_message"

Check the status of the repository
git status

History

Check complete history
git log

One line Histories
git log --pretty=oneline

Controlling which entries are displayed
git log --pretty=oneline --max-count=n
git log --pretty=oneline --since='5 minutes ago'
git log pretty=oneline until='5 minutes ago'
git log --pretty=oneline --author=author_name
git log --pretty=oneline --all
git log --all --pretty=format:'%h %cd %s (%an)' --since='7 days ago'
git log --pretty=format:'%h %ad | %s%d [%an]' --graph --date=short

Tagging Versions

Tagging version 1
git tag v1 (v1 is any tag_name)

Tagging Previous Versions
git checkout v1^
git tag v1-beta (v1-beta is any tag_name of previous version)

Checking out file by Tag Name
git checkout v1   ;   cat filename
git checkout v1-beta   ;   cat filename

Viewing tags
git tag

Viewing tags in the log
git hist master --all

Undoing changes

Undoing local changes(Before staging)
git checkout filename.ext
git status

Undoing staged changes(Before committing)
git reset HEAD filename.ext
git checkout filename.ext
git status

Undoing committed changes
git revert HEAD --no-edit
--no-edit is necessary to generate the output without opening the editor

Creating and Navigating a Branch

Creating a local branch
git checkout -b branch_name
git status

Switching to a branch
git checkout branch_name

Switching to the master branch
git checkout master

List all the branches
git branch

Merging

Merging master into greet branch
git checkout greet
git merge master
git hist --all

Resolving Merge conflicts

Merging master into greet branch
git checkout greet
git merge master

Resolving conflicts
Open the file and resolve the conflict manually.

Commit the conflict resolutiongit add file_name
git commit -m "Merged conflict resolved"
git hist --all


Aliases

Alias for checkout
git config --global alias.co checkout
git co

Alias for commit
git config --global alias.ci commit
git ci

Alias for status
git config --global alias.st status
git st

Alias for branch
git config --global alias.br branch
git br

Alias for history
git config --global alias.hist 'log --pretty=format:"%h %ad | %s%d [%an]" --graph --date=short'
git hist

Alias for type
git config --global alias.type 'cat-file -t'
git type

Alias for dump
git config --global alias.dump 'cat-file -p'
git dump

Getting Older Versions

Check complete history
git hist (if alias set)  or   git log

Checkout previous versions
git checkout commit_hash

Checkout the contents of file
cat filename

Return to latest version in master branch
git checkout master

Removing commits from a branch

Check our History
git hist or git log

First, mark the latest commit
git tag t1 t1 is tag_name

Reset to before t1
git reset --hard t1

Check the history
git hist or git log

Removing the tag

Removing the tag t1
git tag -d t1

Check history
git hist --all or git log --all

Ammending commits

Amend the previous commit
git commit --amend -m "commit_message"

Review history
git hist

Moving Files

Moving files to a directory
mkdir dir_name
git mv filename.ext dir_name
git status
git commit -m "commit_message"

Rebasing

Rebasing master into greet branch
git checkout greet
git rebase master
git hist

Remote Repositories

To clone from a remote repo
git clone http_url/ssh_url

To push from local repo to a remote repo
git push -u origin master

To pull changes from a remote repo
git pull origin master

Fetch changes from remote repo

Check your remote repo
git remote -v

Set a new repo
git remote add upstream http_url

View your remote repo
git remote -v

Fetch changes
git fetch upstream

List all local and remote branches
git branch -va

Merge the changes
git checkout master
git merge upstream/master

Commit and push to new local repo
git commit -m "Merged changes"
git push origin master
